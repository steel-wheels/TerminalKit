# Keybind

The keybind with control key is defined based on readline.

Reference: https://www.gnu.org/software/bash/manual/bash.html#Readline-Interaction

## Cursor movement

C: Control Key. Ex) <code>C-b</code> means "b" with control

|Key    | Description |
|---    | --- |
|C-b    |Move back one character. |
|C-f    |Move forward one character. |
|C-a    |Move to the start of the line. |
|C-e    |Move to the end of the line. |
